## Installation
### Prerequisites
- Node.js (v16+ recommended)
- npm or yarn

### Steps
1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/gavilan-shop.git
   cd gavilan-shop
   ```
2. Install dependencies:
   ```sh
   npm install
   # or
   yarn install
   ```
3. Start the development server:
   ```sh
   npm run dev
   # or
   yarn dev
   ```
4. Open `http://localhost:3000` in your browser.
